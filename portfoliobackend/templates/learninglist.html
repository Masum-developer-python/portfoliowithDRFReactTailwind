15-Day Learning Plan with Resources

  • Day 1: Introduction to Django
    • Install Django and set up a virtual environment: Django Installation Guide
    • Create a basic Django project and app: Django Getting Started
  • Day 2: Django Models and Migrations
    • Learn about models: Django Models Overview
    • Working with migrations: Django Migrations Guide
    • Admin interface basics: Admin Site
  • Day 3: Django Views and Templates
    • Learn about views: Django Views
    • Template basics: Django Templates
  • Day 4: Introduction to Django Rest Framework (DRF)
    • Setting up DRF: Quickstart Tutorial
    • Basic API overview: DRF Official Site
  • Day 5: DRF Serializers
    • Understanding serializers: Serialization in DRF
    • Writing custom serializers.
  • Day 6: Authentication and Permissions in DRF
    • Learn about authentication: Authentication Guide
    • Implement permissions: Permissions
  • Day 7: Testing and API Integration
    • Connecting to databases: Django Database API
    • Testing endpoints: Testing APIs
  • Day 8: Introduction to React
    • Set up React with Create React App: React Docs
    • Understand components: First Component
  • Day 9: React State and Props
    • Learn about props: Props Guide
    • Understand state: State Guide
  • Day 10: Integrating React with Django API
    • Fetching data from APIs: React Effects
    • Axios for API requests: Axios Documentation
  • Day 11: Introduction to Tailwind CSS
    • Set up Tailwind CSS: Tailwind Installation Guide
    • Explore utility classes: Utility-First Classes
  • Day 12: Building Layouts with Tailwind CSS
    • Responsive design: Responsive Design Guide
    • Grid and flexbox utilities: Grid Layout, Flexbox
  • Day 13: Styling React Components with Tailwind CSS
    • Using Tailwind in JSX.
    • Integrating Tailwind with forms.
  • Day 14: Building a Full-Stack Project
    • Combine Django API with React frontend.
    • Style the frontend with Tailwind CSS.
  • Day 15: Deployment
    • Deploy Django with a production server: Deployment Guide
    • Host React frontend (Netlify, Vercel, or similar): Vercel Docs
